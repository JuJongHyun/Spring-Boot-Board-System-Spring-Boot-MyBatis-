# Spring Boot 게시판 웹 애플리케이션

Spring Boot와 MyBatis를 기반으로 게시판 CRUD와 파일 업로드/다운로드 기능을 구현하고,
날짜 기반 파일 관리 구조를 적용한 웹 애플리케이션입니다.

---

기술 스텍
- Backend: Java 21, Spring Boot 3.5
- Template Engine: Thymeleaf
- Frontend: JavaScript, jQuery
- Database: MySQL
- ORM: MyBatis
- Build Tool: Gradle
- Logging: log4jdbc
- Other: Lombok, AOP

---

주요 기능
- 게시판 CRUD
- 파일 업로드 및 다운로드 기능 구현
- 날짜 기반 디렉토리 구조로 파일 관리
- UUID 기반 파일명 생성으로 파일명 중복 방지
